1.爬楼梯是一个典型的递归处理问题。这种问题一般分为三步：1.初始化数据 2.缓存处理 3.递归逻辑  
```
【递归模板】
public void recur(int level, int param) {
     // terminator
     if (level > MAX_LEVEL) {
       // process result
        return; 
     }
     // process current logic
     process(level, param);
     // drill down
     recur( level: level + 1, newParam);
     // restore current status
}
```
